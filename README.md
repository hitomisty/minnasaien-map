# みんなの菜園 配置図マップ

コミュニティガーデン「みんなの菜園」（有明ガーデン）の
コンテナ配置図Webアプリ。菜園の看板に掲示したQRコードから開き、
コンテナをタップすると今植えている植物の名前が表示されます。

**公開ページ**: https://hitomisty.github.io/minnasaien-map/

## 特徴

- アプリのインストール不要。QRコードを読むだけでブラウザで開けます
- カメラ・位置情報・Cookie・外部送信は一切使いません。
  誰でも安心して使えます
- 入口（看板）が画面の下、通路の奥が上。実際に歩く向きと
  スクロール方向が一致する構成です

## 仕組み

- `index.html` + `data.json` の2ファイルのみ。
  フレームワーク・ビルドツール不使用の素のHTML/CSS/JavaScript
- `data.json` に全要素(コンテナ・プランター・固定物)の座標と
  植物名を持ち、`index.html` が読み込んでSVGで描画します
- ホスティングは GitHub Pages

## 更新方法(植え替えのとき)

1. `data.json` を開く
2. 該当コンテナの `plants` の配列を書き換える
3. `lastUpdated`(最終更新日)を今日の日付に書き換える
4. コミットしてpushすると数分でページに反映されます

※ 座標やレイアウトを触る必要はありません。植物名と日付だけ。

- このリポジトリおよびページには、植物情報と菜園の設備情報のみを
  掲載します。個人名・連絡先・当番などの情報は掲載しません
